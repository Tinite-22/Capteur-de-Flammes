Un capteur de flamme est un dispositif électronique utilisé pour détecter la présence d’une flamme ou de feu. 
Il fonctionne généralement en analysant la lumière infrarouge (IR) émise par le feu. Lorsqu’une flamme est détectée, 
le capteur envoie un signal numérique ou analogique à un microcontrôleur (comme Arduino), ce qui permet d’activer une alarme, 
couper un circuit ou lancer une action de sécurité.
